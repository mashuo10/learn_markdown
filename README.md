### 使用技巧

#### gitignore
1. gitignore包含 *.png
 - 远端+png文件，近端+png文件
 - 远端-png文件，近端-png文件
 - 近端+png文件，远端不更新
  - 近端-png文件，远端不更新
2.  gitignore不包含 *.png
 - 远端+(-)png文件，近端+(-)png文件
 - 近端+(-)png文件，远端+(-)png文件
